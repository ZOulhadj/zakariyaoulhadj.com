Vulkan Model Viewer and Exporter (VMVE). A standalone application developed as
part of my final year project as University which aims to provide an
easy-to-use, efficient platform for 3D graphics rendering and asset
encryption. It serves as an important tool and as a stepping stone that aims to
bridge to gap between new users and industry-leading rendering engines.  To
achieve this, VMVE provides a small but specific subset of features that are
designed from the ground up to be easy to use and understand for rapid graphics
prototyping such as constructing virtual environments, simulating the effect of
light on 3D geometry and securing digital assets through the use of encryption.
The project was accompanied with a [paper](/portfolio/report.pdf) titled
"Optimizing workflow for virtual environments using Vulkan Model Viewer and
Exporter".
